About the Project

BrandHex was built to solve the repetitive task of inspecting source code to find high-quality company logos and brand colors. By leveraging the Microlink API, this tool automates the extraction of structural brand metadata globally.

Tech Stack

  • HTML5 / CSS3 (Grid & Flexbox)
  • Vanilla JavaScript (ES6+)
  • Microlink API Integration
  • Vercel Edge Network Deployment
